**Web Project : Online Coffee Ordering Website**

**Created By:** Shani Patel, Saurabh Tripathi
**College:** SHEAT College of Engineering
**Date:** 12th November 2025

---

### **1 (Project Overview)**

* A web-based platform to order coffee online.
* Built using HTML, CSS, and JavaScript.
* Focused on user-friendly design and smooth animations.
* Includes: Login → Shop Now → Order → Order Successful flow.

---

### **2 (Website Pages)**

* **Login Page:** Multi-step signup form.
* **Shop Now Page:** List of coffee types with prices & images.
* **Order Page:** Customer enters name, address, coffee type, etc.
* **Order Successful Page:** Confirmation animation with delivery tracker.

---

### **3 (Key Features)**

(a) Responsive web design (works on all devices)
(b) Interactive UI with hover effects
(c) JavaScript form validation
(d) Coffee theme colors & animations
(e) Simple and smooth navigation

---

### **4 (Conclusion)**

* Demonstrates practical use of front-end web development.
* Realistic design of an e-commerce style website.
* “Enjoy every moment with a cup of code & coffee.”
* **Future Add-ons:** Payment Gateway, Database, Login System.
